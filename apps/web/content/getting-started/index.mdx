Welcome to Claude Code! This guide will help you get up and running with Anthropic's powerful AI coding assistant. Whether you're a solo developer or part of a team, Claude Code can dramatically accelerate your development workflow.

## What is Claude Code?

Claude Code is a command-line interface (CLI) tool that brings Claude AI directly into your development environment. It can:

- **Read and understand your codebase** - Claude analyzes your project structure, code patterns, and documentation
- **Write and modify code** - Generate new features, fix bugs, and refactor existing code
- **Execute commands** - Run builds, tests, and other development tasks
- **Explain complex code** - Get clear explanations of unfamiliar code or concepts

## Quick Start

Get Claude Code running in just two steps:

```bash
# Install Claude Code globally
npm install -g @anthropic-ai/claude-code

# Start your first session
claude
```

## Key Features

| Feature | Description |
|---------|-------------|
| **Agentic Coding** | Claude autonomously plans and executes multi-step tasks |
| **Project Awareness** | Understands your entire codebase context via CLAUDE.md |
| **Safe Execution** | Requires approval before making file changes |
| **Multi-Model Support** | Choose between Claude Sonnet, Opus, and Haiku |
| **MCP Integration** | Extend capabilities with Model Context Protocol servers |
